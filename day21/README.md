## Tests

I wrote some tests to help me nail down the logic.  You can run them with:

```bash
./bin/rspec
```

Or see the full list of tests with:

```bash
./bin/rspec --format documentation
```

> You may need to install RSpec with:
> 
> ```bash
> bundle install
> bundle binstubs --all
> ```
